# Ví điện tử sử dụng Blockchain
- Sử dụng thư viện Secp256k1 để tạo ví tiền và các thao tác với khóa.
- Thực hiện giao dịch, ký nhận giao dịch và xem lịch sử giao dịch.
- Cho thấy sơ bộ luồng hoạt động của Blockchain trong giao dịch.
